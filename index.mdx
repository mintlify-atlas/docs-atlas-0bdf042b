## Get up and running

<Steps>
  <Step title="Clone and install dependencies">
    Clone the repository and run `npm install` to install all frontend and backend dependencies.
  </Step>
  <Step title="Add your API key">
    Create a `.env` file in the project root and add your `PERPLEXITY_API_KEY`. See [Environment Setup](/setup/environment) for all required variables.
  </Step>
  <Step title="Start the backend server">
    Run `npm run server` to start the Express backend on port 3001. The AI endpoints will be available immediately.
  </Step>
  <Step title="Start the frontend">
    In a separate terminal, run `npm run dev`. The app opens at `http://localhost:5173` with all features live.
  </Step>
</Steps>

<Note>
  You can run both the frontend and backend simultaneously with `npm run dev:all`.
</Note>
